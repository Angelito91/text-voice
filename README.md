# Python Email
una aplicación para pasar de texto a voz


## Instalación
Debes tener:
* Python 3.x
* venv

```
python -m venv .venv
```

```
pip install -r requirements.txt
```

## Ejecutar
En la terminal ejecuta este comando:

> [!IMPORTANT]
> Debes tener siempre un archivo data.txt y con datos en la raiz del proyecto

```
python main.py
```
